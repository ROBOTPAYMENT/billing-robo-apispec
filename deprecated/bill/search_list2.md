# 請求書参照2

`/api/v1.0/bill/search_list2`

請求書の参照処理を実行します。
請求書参照とは、レスポンスについて単価・数量・消費税の型が異なります。

## リクエスト
- Method URL: `https://billing-robo.jp:10443/api/v1.0/bill/search_list2`
- Preferred HTTP method: `POST`
- Accepted content types: `application/json`
- Encode: `UTF-8`
